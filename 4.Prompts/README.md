## 🎯 What We Built Overall

In this module, we built two main types of applications to demonstrate prompting techniques:
1. **Dynamic Prompting System:** We built a reusable `PromptTemplate` to generate structured instructions for an LLM (e.g., summarizing research papers) and integrated it into a **Streamlit UI** (`prompt_ui.py`) for an interactive and user-friendly web experience.
2. **Context-Aware Chatbots:** We moved beyond single-turn interactions to build conversational agents that remember past interactions. We implemented an interactive terminal-based chatbot (`chatbot.py`) that maintains conversational state across a session.

## 🛠️ What We Used & Learned

### 1. Prompt Templates (`prompt_generator.py`, `prompt_ui.py`)
Instead of hardcoding strings, we use templates to dynamically inject user input and ensure consistent instruction formatting.

*   **Concepts Learned:** Prompt engineering, reusability, variable injection, and separating prompt logic from application code.
*   **Key Classes/Methods:**
    *   `PromptTemplate`: The base class for creating string prompts.
    *   `input_variables`: Defines the placeholders (e.g., `["topic", "style"]`).
    *   `template.format(**kwargs)`: Fills the placeholders with actual values at runtime.
    *   `template.save("template.json")`: Serializes the template to disk so it can be version controlled and loaded without code changes.
    *   `load_prompt("template.json")`: Deserializes a saved template for reuse.

**Example:**
```python
from langchain_core.prompts import PromptTemplate

template = PromptTemplate(
    template="Explain {topic} in a {style} way.",
    input_variables=["topic", "style"]
)
prompt = template.format(topic="Quantum Physics", style="poetic")
```

### 2. Chat Messages (`Chatbot/messages.py`)
Chat models use specialized message types rather than raw strings to represent different roles in a conversation. This provides better structure and steerability for modern LLMs.

*   **Concepts Learned:** Structuring conversations using distinct roles.
*   **Key Types:**
    *   `SystemMessage`: Instructions or context given to the model to set its behavior (e.g., "You are a helpful assistant").
    *   `HumanMessage`: The input or question provided by the user.
    *   `AIMessage`: The response generated by the LLM.

### 3. Chat Prompt Templates (`Chatbot/chat_prompt_template.ipynb`)
For chat models, we use templates that generate a sequence of structured messages instead of a single string prompt.

*   **Concepts Learned:** Formatting complex multi-turn prompts dynamically.
*   **Key Classes/Methods:**
    *   `ChatPromptTemplate`: Creates a template for a list of messages.
    *   `ChatPromptTemplate.from_messages([...])`: Constructs the template from a mix of string tuples (representing roles) and placeholder objects.
    *   `MessagesPlaceholder(variable_name="chat_history")`: A special placeholder used to inject a dynamic list of previous messages (the conversation history) into the prompt structure.

### 4. Interactive Chatbots (`Chatbot/chatbot.py`)
Building an application loop that continuously takes user input, appends it to a history list, calls the LLM, and appends the response, creating a fluid conversation.

*   **Concepts Learned:** Managing conversational state and context windows.
*   **Workflow:**
    1. Initialize a `chat_history` list, usually starting with a `SystemMessage`.
    2. Get user input and append it as a `HumanMessage`.
    3. Send the entire `chat_history` to the Chat Model.
    4. Receive the response and append it as an `AIMessage`.
    5. Repeat the cycle.

## 🚀 Quick Start

Run the following commands from the root of the project to test the applications:

**1. Generate the Prompt Template:**
```bash
python 4.Prompts/prompt_generator.py
```

**2. Run the Streamlit UI:**
```bash
streamlit run 4.Prompts/prompt_ui.py
```
*(Tip: In `prompt_ui.py`, consider using premium fonts like **Plus Jakarta Sans** via Google Fonts for a modern, clean UI appearance.)*

**3. Start the Interactive Chatbot:**
```bash
python 4.Prompts/Chatbot/chatbot.py
```
*(Type your queries and type 'exit' or 'quit' to close the chatbot session.)*
